2.  useState
    use for state update or change.

   -> How to use ?

    const [enteredTitle,setEnteredTitle] = useState('Initial value');

    // Initial value => Initial value of state.
    // enteredTitle => variable that store value.
    // setEnteredTitle => function that set state value.

   -> Update single value in state

    const [enteredTitle,setEnteredTitle] = useState('Initial value');
    setEnteredTitle('updated value');

   -> Update multiple value in state
    
      const [userInput, setUserInput] = useState({

     setUserInput({
